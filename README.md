# 🚀 Projects

### ☕ Rabuste Coffee
Full-stack platform built for a **real café in Surat** to manage online orders, workshops, and loyalty rewards without relying on third-party platforms.

**Tech:** Next.js • TypeScript • Supabase • PostgreSQL • Prisma • Razorpay

---

### 📦 CoreInventory
Modular **Inventory Management System** built during a hackathon to digitize warehouse operations.

Features real-time stock tracking, multi-warehouse support, and a complete stock ledger for audit trails.

**Tech:** React • Node.js • Express • SQLite • JWT

---

### 📅 NextEventNow
Campus event discovery platform created to remove the **“scroll through WhatsApp groups” problem**.

Aggregates academic and cultural events into one searchable dashboard with categories, form links, and event memories.

**Tech:** React • Node.js • MongoDB

---

### 🩺 Sehat Mitra
AI-powered healthcare assistant that analyzes user-described symptoms and generates preliminary recommendations using **NLP + machine learning**.

Built during **MakerNova Hackathon** with a Flask backend and ML models.

**Tech:** Python • Flask • NLP • Scikit-learn • React

---

### ✅ DoneDay
Habit tracking web application designed to help users stay consistent with daily routines.

Tracks habits, shows weekly progress tables, and calculates completion percentages.

**Tech:** HTML • CSS • JavaScript • Node.js
